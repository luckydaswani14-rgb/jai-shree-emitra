# Jai Shree E-Mitra 🇮🇳

### Your Trusted Digital Service Center — RK Puram B, Kota, Rajasthan

A **premium, responsive, bilingual (English & Hindi)** website for **Jai Shree E-Mitra**, a government-authorized digital service and documentation center in Kota, Rajasthan, India.

---

## 🌐 Live Website

Open `index.html` directly in any browser — no server required!

---

## ✨ Features

- 🎨 **Premium Design** — Saffron + Blue + White theme, Glassmorphism cards, smooth animations
- 🌙 **Dark / Light Mode** toggle
- 🌐 **Bilingual** — English & Hindi language switch (all content translates instantly)
- 📋 **14+ E-Mitra Services** — Electricity Bill, Aadhaar e-KYC, Pension e-KYC, RTE Forms, Birth/Death/Caste/Domicile/Income Certificates, Jan Aadhaar, PAN Card & more
- 🔍 **Documents Checker** — Select any service to see required documents & processing time
- 📣 **Government Schemes Board** — Live notification banner + scheme cards
- 📦 **Application Tracker** — Enter receipt number (e.g. `EM-554901`) to check real-time status
- ⭐ **Customer Reviews** — Submit, approve, and display verified reviews
- 📅 **Appointment Booking** — Online slot request form
- 💬 **Live Support Chat** — Simulated bilingual chatbot assistant
- 📥 **Download Forms** — Caste, Domicile, Income, Jan Aadhaar, Birth, RTE form templates
- 🖼️ **Lightbox Gallery** — Office, workstation, customer service, licenses
- ⚙️ **Admin Dashboard** — Secured panel to manage reviews, schemes, tracking IDs, appointments
